# Product Modeling

## Objective

Turn platform vision into a concrete product model.

## Core Outputs

- platform positioning
- user personas
- organization model
- team model
- department model
- project space model
- collaboration model
- visibility model
- mode system

## Working Rules

- Define product entities before technical services.
- Keep user value explicit.
- Separate long-term vision from v1 product behavior.
- Make relationships between users, organizations, teams, and projects unambiguous.

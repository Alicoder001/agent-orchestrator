# Workspace Experience

## Objective

Design the platform's experience architecture with clear operational value.

## Core Outputs

- workspace shell
- digital office model
- 2D workspace patterns
- 3D strategy
- chat and command surface
- operator dashboard model
- mode experience mapping

## Working Rules

- Prioritize operational clarity over visual novelty.
- Treat 2D as the practical baseline unless proven otherwise.
- Keep the dashboard and workspace shell complementary.
- Make experience choices traceable to user jobs.

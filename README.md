# Agent Orchestrator

Ushbu repository yangi, professional darajadagi monoreponi noldan qurish uchun tayyorlanmoqda.

## Hozirgi holat

- `legacy-monorepo/` oldingi tizimning arxiv nusxasi bo'lib, faqat referens sifatida saqlanadi.
- `docs/` yangi platformaning asosiy hujjatlashtirish markazi hisoblanadi.
- Yangi arxitektura, product modeli va platforma qarorlari avval hujjatda shakllantiriladi, keyin kodga o'tiladi.

## Asosiy tamoyillar

- Legacy kod yangi tizimga bevosita bog'lanmaydi.
- Yangi monorepo docs-first va architecture-first yondashuvda quriladi.
- Chegaralar, ownership, workflow va platforma qoidalari boshidan aniq yoziladi.

## Hujjatlar

Ishni boshlash uchun birinchi navbatda quyidagilarni ko'ring:

- `docs/README.md`
- `docs/00-foundation/`
- `docs/01-architecture/`
- `docs/05-governance/`

## Legacy qatlami

`legacy-monorepo/` ichida oldingi implementatsiya to'liq saqlangan. U ishlab turgan aktiv tizim emas, balki:

- arxitektura va UX referensi
- migration uchun tarixiy manba
- qayta foydalanish mumkin bo'lgan patternlar ombori

Yangi tizim shu arxivdan ilhom oladi, lekin to'g'ridan-to'g'ri uning ustiga qurilmaydi.

# Architecture Decision Process

## Purpose

Define how major technical decisions are proposed, reviewed, approved, and recorded.

## Decision Flow

1. Problem is written clearly.
2. Constraints and options are listed.
3. Proposed decision is reviewed.
4. Approved decision becomes an ADR.
5. Implementation follows the ADR unless superseded.

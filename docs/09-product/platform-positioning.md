# Platform Positioning

## Maqsad

Ushbu hujjat platformaning bozordagi o'rni, raqobatchilardan farqi va qiymat taklifi (value proposition) ni belgilaydi.

Bu hujjat `vision.md`, `business-context.md`, `platform-thesis.md` va `scope.md` ga asoslanadi.

---

## One-liner

> Agent Orchestrator — AI-native digital tashkilotlar uchun operatsion platforma.

---

## Problem Statement

Hozirgi AI coding tool'lar (Cursor, GitHub Copilot, Windsurf) **bitta developer + bitta agent** model'ida ishlaydi. Lekin tashkilotlarga kerak:

| Muammo | Hozirgi holat | Bizning yechim |
|--------|--------------|---------------|
| Ko'p agent boshqarish | Terminal'da qo'lda | Dashboard + attention system |
| Agent'lar jamoaviy ishi | Alohida tool'lar | Organization → Team → AgentSlot |
| Sifat nazorati | Yo'q | Mode system + human gates |
| Kuzatuv (observability) | Yo'q | Session tracking + event log |
| Tashkiliy tuzilma | Yo'q | Org → Department → Team model |

---

## Value Proposition

### Solo Developer uchun

> "Barcha AI agent'laringizni bitta joydan boshqaring. Qachon diqqat kerak bo'lsa — platforma aytadi."

### Tech Lead uchun

> "Jamoangizning AI agent'larini sprint board'dek kuzating. Sifatni yo'qotmasdan tezlikni oshiring."

### Startup CTO uchun

> "50+ AI agent'ni 5 kishilik jamoa bilan boshqaring. Human-AI ratio'ni 1:10 ga olib boring."

---

## Category Definition

**Yangi kategoriya:** "AI Workforce Platform" yoki "Agent Orchestration Platform"

Bu mavjud kategoriyalardan farqli:

| Kategoriya | Misol | Biz bilan farq |
|-----------|-------|----------------|
| AI Coding Assistant | Cursor, Copilot | Individual tool, org structure yo'q |
| DevOps Platform | GitHub Actions, GitLab CI | Pipeline-centric, agent management yo'q |
| Project Management | Jira, Linear | Human-only, AI agent integration yo'q |
| AI Agent Framework | LangChain, CrewAI | Developer framework, end-user platform emas |

**Positioning:** Biz — bu tool'larni birlashtiruvchi **orchestration layer**.

---

## Competitive Differentiation

| Differentiator | Tavsif |
|----------------|--------|
| **Org-native** | Organization → Department → Team → Agent tuzilmasi built-in |
| **Multi-agent** | Bir vaqtda 50+ agent parallel |
| **Attention system** | Inson diqqatini to'g'ri boshqarish |
| **Mode system** | Supervised → Collaborative → Autonomous evolyutsiya |
| **Agent-agnostic** | Cursor, Copilot, Claude — istalgan agent |
| **Self-hostable** | Open-core, o'z server'ida ishlash mumkin |

## Target Market

### Primary (V1)

- AI-first startuplar (1-50 developer)
- Freelancer va solo developer'lar (multiple projects)
- Small tech team'lar AI adoption boshlaganlar

### Secondary (V2-V3)

- O'rta kompaniyalar (50-200 developer)
- Consulting firmalar (multi-client agent management)

### Tertiary (V5+)

- Enterprise (200+ developer)
- Government va regulated industries

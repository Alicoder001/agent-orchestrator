# Workspace Studio

Unified 2D and 3D workspace shell for Agent Orchestrator.

## Purpose

- premium navigation shell
- organization and project switching
- 2D office workspace
- 3D synchronized environment
- docked orchestration chat

## Run

```bash
pnpm --filter @composio/ao-workspace-studio dev
```

Default local URL:

```text
http://127.0.0.1:4174
```
